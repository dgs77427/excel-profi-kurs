## Warum SVERWEIS allein oft nicht reicht

Hier ein kurzer Einschub, weil das oft missverstanden wird: SVERWEIS ist die bekannteste Suchfunktion in Excel, aber sie hat eine harte Einschränkung — sie kann nur nach rechts suchen. Willst du einen Wert links vom Suchkriterium finden, bist du mit SVERWEIS am Ende.

Genau da kommen INDEX und VERGLEICH ins Spiel. Die Kombination ist flexibler: INDEX gibt den Wert an einer bestimmten Position zurück, VERGLEICH findet die Position. Zusammen können sie in beide Richtungen suchen und sind robuster gegen Tabellenstrukturänderungen. Microsoft selbst empfiehlt in seinem Support-Artikel „Nachschlagen von Werten mit SVERWEIS, INDEX oder VERGLEICH" die Kombination als Alternative, wenn SVERWEIS an seine Grenzen stößt.

Der Kurs erklärt beide Ansätze und zeigt, wann welche Funktion sinnvoller ist. Genau diese Entscheidungskompetenz unterscheidet jemanden, der Excel bedient, von jemandem, der es **beherrscht**.
